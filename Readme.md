# House Price Analysis and Prediction

## Objective
To analyze the relationship between housing features and price using statistical and machine learning techniques.

## Methods Used
- Data Cleaning using Pandas
- Exploratory Analysis
- Linear Regression Model
- Model Evaluation using Mean Squared Error

## Tools
Python, Pandas, Matplotlib, Scikit-learn

## Outcome
Built a simple and interpretable predictive model to understand price trends.

## Learning
This project demonstrates practical understanding of:
- Data preprocessing
- Model training/testing workflow
- Statistical interpretation of results
